# Evergreen Elegance Jewelry - E-commerce

A premium gold jewelry e-commerce platform built with React, Node.js, and MongoDB.

## Features
- **Luxury UI**: Stunning black and gold theme with Playfair Display and Outfit fonts.
- **Product Management**: Category-wise filtering, search, and detailed product specs.
- **Shopping Flow**: Easy Cart and Wishlist management.
- **Secure Checkout**: Integrated with Razorpay (Mocked).
- **Admin Panel**: Full CRUD operations for products and order tracking.
- **WhatsApp Integration**: Direct inquiry button on product pages.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.

## Tech Stack
- **Frontend**: React.js, Vite, Framer Motion, Lucide Icons.
- **Backend**: Node.js, Express, Mongoose, JWT, Razorpay SDK.
- **Database**: MongoDB.

## Getting Started

### Prerequisites
- Node.js installed
- MongoDB installed and running

### Installation

1. **Clone the repository**
2. **Setup Server**
   ```bash
   cd server
   npm install
   # Configure .env with your MONGODB_URI and JWT_SECRET
   npm start
   ```
3. **Setup Client**
   ```bash
   cd client
   npm install
   npm run dev
   ```
